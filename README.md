## 使用注意事项

狗云虽然有不少优点，但也有几个坑需要注意：

### 1. 余额欠费处理严格

**超过3天欠费，数据直接删除，没有任何保留**。

这点真的要小心。建议开启余额预警，或者绑定自动充值。很多人反馈说一不留神欠费了，数据就没了，找客服也没用。

### 2. 退款限制

- 充值时间不超过一个月
- 商品必须可销毁（购买页面会标注）
- 不要点"完结"按钮，否则无法退款

所以建议第一次购买先少充点钱，测试满意再继续。

### 3. 稳定性有待观察

狗云是小厂，虽然口碑不错，但毕竟不是阿里云、腾讯云那种级别。线路偶尔会抽风，MG机房的BGP线路据说经常断线或者切换路由。

如果你对稳定性要求特别高，建议选香港KC机房或者美国CN2线路，这两个相对靠谱一些。

### 4. 流量转存功能

弹性云的流量可以转存到下个月，这点倒是挺良心的。但要注意：转存有上限，别囤太多。
